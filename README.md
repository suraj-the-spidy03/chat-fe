# 💬 Real-Time Chat App (Frontend)

A modern real-time chat application built with **Next.js, Socket.IO, and Tailwind CSS**.  
Supports live messaging, user separation, and responsive UI with dark theme.

---

## ⚡ Features

- 🔥 Real-time messaging (Socket.IO)
- 👤 User-based message separation
- 🌙 Dark Snapchat-style UI
- 📱 Fully responsive design
- ⌨️ Send message using Enter key
- 🔄 Auto-scroll to latest message
- 🧹 Refresh chat after 100 messages
- 💅 Smooth animations & modern UI

---

## 🛠️ Tech Stack

- **Next.js (App Router)**
- **React.js**
- **Socket.IO Client**
- **Tailwind CSS**
- **TypeScript**
